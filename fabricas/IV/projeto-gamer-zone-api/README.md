## Description

---
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

```

## Test

```bash
# unit tests
$ yarn test

# test coverage
$ yarn test:cov
```
